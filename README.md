# Cursor Useful Commands

Collection of useful Cursor IDE commands and workflows.

## Purpose

This repository contains custom Cursor commands and workflows that enhance development productivity.

## Commands

### Linear Integration

- **linear-from-prompts**: Create Linear issues from Cursor chat/prompt history
- **linear-from-commits**: Create Linear issues from git commit history

## Usage

These commands are designed to be used within Cursor IDE. Place command files in `.cursor/commands-private/` directory of your project.

## Contributing

Add new useful commands by creating markdown files following the existing command structure.
